📅 Resumo de Códigos Básicos: Guia Rápido de HTML

Para ajudar todo mundo a colocar a mão na massa no laboratório, aqui está o nosso guia de sobrevivência HTML. Use essas tags para estruturar o seu site!

  • <h1> até <h6>: 🏷️ Títulos e subtítulos. O <h1> é o principal e maior da página.
  • <p>: ✍️ Parágrafo de texto comum.
  • <strong>: ⚠️ Deixa o texto em negrito para dar destaque.
  • <em>: ✨ Deixa o texto em itálico para dar ênfase.
  • <ul>: 📋 Cria uma lista com marcadores (bolinhas).
  • <ol>: 🔢 Cria uma lista numerada automática (1, 2, 3...).
  • <li>: 🔹 Cada item individual de uma lista (deve ficar dentro de ul ou ol).
  • <br>: ↩️ Quebra de linha simples (não precisa de tag de fechamento).

💡 Dica de ouro: Lembre-se sempre de abrir a tag <tag> e fechar no final do texto usando a barra </tag>!

💡 Dica de Web Design: Como centralizar imagens com CSS

Muitas vezes, quando usamos a propriedade text-align: center; no CSS, ela funciona muito bem para textos, mas não mexe na nossa imagem. Para resolver esse problema de forma isolada e deixar o nosso mascote perfeitamente no meio da página, aprendemos um truque muito importante!


img {
  width: 400px;
  height: 400px;
  display: block; /* Transforma a imagem em bloco */
  margin: 0 auto; /* Alinha os lados automaticamente */
}

🛠️ Entendendo as propriedades:

  • display: block;: Por padrão, as imagens ficam na mesma linha dos textos. Ao mudar para bloco, dizemos ao navegador que a imagem deve ocupar a sua própria linha exclusiva no site.
  • margin: 0 auto;: O valor auto indica para o navegador calcular o espaço restante nas laterais esquerda e direita de forma igual, empurrando o elemento exatamente para o centro da tela.

Agora que dominamos esse truque, nossos layouts vão ficar muito mais organizados e profissionais! Testem nos seus projetos individuais e vejam a mágica acontecer.

🗺️ Mapa do Nosso Código: Estrutura Básica do Blog

Para quem está organizando o arquivo ou com dificuldades no VS Code, confiram o esquema visual de como as tags devem estar encaixadas dentro do seu documento:

<!DOCTYPE html>
<html lang="pt-br">
<head>
<meta charset="UTF-8">
<title>Título do seu Blog</title>
<style> /* Estilização aqui */ </style>
</head>
<body>
<header> /* Topo do site: Nome do blog e introdução do tema */ </header>
<main> /* Conteúdo principal: Imagem ilustrativa e os 3 posts */ </main>
</body>
</html>

✅ Checklist de Atividades: O que deve ser entregue?

Verifique item por item no seu projeto para garantir que o seu grupo cumpriu todos os requisitos obrigatórios:

  • 🔲 Tema Livre: Definição clara do tema escolhido pelo grupo (pode ser sobre jogos, esportes, música, tecnologia, hobbies, etc.).
  • 🔲 Estrutura Organizada: Uso correto das tags estruturais <header> e <main> dentro do <body>.
  • 🔲 Conteúdo Mínimo: Pelo menos 3 postagens completas desenvolvendo o assunto do seu blog.
  • 🔲 Imagem Temática: Inserção de pelo menos 1 imagem configurada no HTML que seja totalmente relacionada ao tema do seu site.
  • 🔲 Hospedagem e Editor: Projeto devidamente sincronizado entre o GitHub e o VS Code local.
